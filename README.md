# Type-Utils（WIP）

使用 Typescript 开发的一套基于类型系统的工具集（Just For Fun）。

## 如何使用

> 本项目中的代码会有两个版本，分别是：
>
> - **完全使用类型工具二次开发**的纯娱乐版本。
> - **优化泛型递归次数**后可一定程度中引入项目的生产版本（降低了可读性）。

本项目可以用于学习 Typescript 的类型编程，但不推荐在项目中引用。

如果你确实需要使用它，请遵守以下条件：

- 只用于创建不涉及**过多**递归的类型（因为 typescript 中对于泛型参数调用次数的限制）
